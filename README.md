**Katakana for Cyrillic.** Chuzhditsa (Bulgarian *чуждица*, "foreignism") is a designed
extension of the Cyrillic alphabet for writing foreign words phonetically — and a typeface
that marks them visually. Japanese sets loanwords apart in katakana: a parallel register
whose shapes alone say "this word is a guest," with adaptation rules deterministic enough
that two writers converge on one spelling. Cyrillic has no equivalent, so *lunch* becomes
ланч or лънч and /θ/ collapses into т or с at random. Chuzhditsa supplies the missing
device in three layers: the unchanged **base** alphabet (any Slavic Cyrillic orthography);
**extension letters** for foreign sounds, each derived by a featural grammar (one diacritic
= one phonetic feature — the hook of щ generalized to қ ҳ ң ҫ ҙ); and optional **prosody**
marks for stress, tone and length. The register is marked by the typeface itself: monoline
forms with round bowls drawn from Glagolitic — the oldest Slavic shapes marking the newest
words. It marks words, never speakers, and a loan that naturalizes graduates back to the
plain alphabet.

Current version: **2.1**. Start with the [plain-language whitepaper](WHITEPAPER.md), the
[English specification](CHUZHDITSA.en.md), or the [journal manuscript](paper/chuzhditsa.pdf).

> Naming note: the system is *Chuzhditsa/чуждица*; this repository's slug (`chuzbitza`)
> predates the final name.

## Installing the fonts

The family ships in four styles (Regular, Bold, Italic, Bold Italic), each as TTF and OTF,
in [fonts/](fonts/). Double-click to install (macOS Font Book / Windows), or copy to
`~/.local/share/fonts` on Linux and run `fc-cache -f`. The extension letters are real
Unicode codepoints (қ ң ҳ ҫ ҙ ў ѕ ѫ ѧ …), so text stays copyable and searchable. Ligature
fusion (нь→њ-form, ьѧ→ѩ) and mark placement need an OpenType-aware renderer — any browser
or modern OS text stack qualifies.

## Building from source

Everything is generated by code — glyphs are parametric monoline strokes, and the
orthography's fusion and anchoring rules are the font's GSUB/GPOS features.

```sh
python3 -m pip install -r tools/requirements.txt

python3 tools/build_font.py    # 1. build all 8 font files into fonts/
python3 tools/gen_docs.py      # 2. regenerate the three main docs + specimen.html (after step 1)
python3 tools/proof.py         # 3. optional: render tools/proof.png (HarfBuzz-shaped proof sheet)
python3 tools/render_hero.py   # 4. optional: regenerate docs/hero*.png

cd paper && xelatex chuzhditsa.tex && xelatex chuzhditsa.tex   # optional: the manuscript (XeLaTeX)
```

`CHUZHDITSA.md`, `CHUZHDITSA.en.md`, `CHUZHDITSA.uk.md` and `specimen.html` are
**generated** by `tools/gen_docs.py` — edit that script, not the outputs. The short
per-language guides are hand-written.

## Name

Bulgarian **чуждица**, romanized **Chuzhditsa** — in purist usage, a foreign word the
language would supposedly be better off without. The name is chosen against that usage:
where purism hands the foreignism an expulsion order, this system hands it a well-made
costume and a seat. Piloted on Bulgarian, generalized to the Slavic Cyrillic superset,
validated externally on Albanian.
