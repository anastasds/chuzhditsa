**Katakana for Cyrillic.** Chuzhditsa (Bulgarian *чуждица*, "foreignism") is a designed
extension of the Cyrillic alphabet for writing foreign words phonetically — and a typeface
that marks them visually. Japanese sets loanwords apart in katakana: a parallel register
whose shapes alone say "this word is a guest," with adaptation rules deterministic enough
that two writers converge on one spelling. Cyrillic has no equivalent, so *lunch* becomes
ланч or лънч and /θ/ collapses into т or с at random. Chuzhditsa supplies the missing
device in three layers: the unchanged **base** alphabet (any Slavic Cyrillic orthography);
**extension letters** for foreign sounds, each derived by a compositional grammar of
recurring diacritic operations (the hook of щ generalized to қ ҳ ң ҫ ҙ); and optional **prosody**
marks for stress, tone and length. The register is marked by the typeface itself: round-bowled forms drawn from Glagolitic —
the oldest Slavic shapes marking the newest words — in a four-face family that includes a
Times-matched cut for citations inside ordinary serif text. It marks words, never speakers, and a loan that naturalizes graduates back to the
plain alphabet.

Current version: **2.5**, with the v3 revision — four families from one skeleton set
(**2b** register face, **Grotesk**, **Serif**, and the Times-matched **Inline** the papers
are set in) — in [fonts/v3/](fonts/v3/), built by [tools/build_font_v3.py](tools/build_font_v3.py).
Start with the [plain-language whitepaper](WHITEPAPER.md), the
[English specification](CHUZHDITSA.en.md), or the [journal manuscript](paper/chuzhditsa.pdf).

Designed and directed by **Anastas Stoyanovsky**; constructed, under his criticism, by
Claude (Anthropic) — the process itself is documented in the process study,
[paper/process.pdf](paper/process.pdf), and reflected on in an annotated op-ed,
[paper/oped.pdf](paper/oped.pdf).

## Installing the fonts

Four families ship (2b, Grotesk, Serif, Inline), each in four styles (Regular, Bold,
Italic, Bold Italic) as TTF and OTF, in [fonts/v3/](fonts/v3/); the original v2.5 single
family remains in [fonts/](fonts/). Double-click to install (macOS Font Book / Windows), or copy to
`~/.local/share/fonts` on Linux and run `fc-cache -f`. The extension letters are real
Unicode codepoints (қ ң ҳ ҫ ҙ ў ѕ ѫ ѧ …), so text stays copyable and searchable. Ligature
fusion (нь→њ-form, ьѧ→ѩ) and mark placement need an OpenType-aware renderer — any browser
or modern OS text stack qualifies.

## Building from source

Everything is generated by code — glyphs are parametric centerline strokes (the v3 engine
adds a tangent-angle contrast function and boolean-unioned outlines), and the orthography's
fusion and anchoring rules are the font's GSUB/GPOS features.

```sh
python3 -m pip install -r tools/requirements.txt

python3 tools/build_font.py    # 1. build all 8 font files into fonts/
python3 -m pip install fonttools uharfbuzz skia-pathops  # build deps (pathops: union-before-emit)
python3 tools/build_font_v3.py # 1b. build the four v3 families (2b/Grotesk/Serif/Inline) into fonts/v3/
#     builds are byte-reproducible: head timestamps are pinned via SOURCE_DATE_EPOCH
#     (set in-script, overridable in the environment) — identical source, identical bytes
python3 tools/gen_docs.py      # 2. regenerate the four language specs + specimen.html (after step 1)
python3 tools/proof.py         # 3. optional: render tools/proof.png (HarfBuzz-shaped proof sheet)
python3 tools/render_hero.py   # 4. optional: regenerate docs/hero*.png

cd paper && xelatex chuzhditsa.tex && xelatex chuzhditsa.tex   # optional: the manuscript (XeLaTeX)
```

`CHUZHDITSA.md`, `CHUZHDITSA.en.md`, `CHUZHDITSA.uk.md`, `CHUZHDITSA.ru.md` and `specimen.html` are
**generated** by `tools/gen_docs.py` — edit that script, not the outputs. The short
per-language guides are hand-written.

## Name

Bulgarian **чуждица**, romanized **Chuzhditsa** — in purist usage, a foreign word the
language would supposedly be better off without. The name is chosen against that usage:
where purism hands the foreignism an expulsion order, this system hands it a well-made
costume and a seat. Piloted on Bulgarian, generalized to the Slavic Cyrillic superset,
validated externally on Albanian.
